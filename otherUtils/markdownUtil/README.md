# Markdown 编辑器 / 浏览器

纯离线、零依赖的单文件 Markdown 编辑器，面向 Chrome 125+，可直接通过 `file://` 打开。

## 功能

- 双栏实时编辑与预览，桌面端可拖动中间分隔条调整为 20%～80%；支持鼠标、触摸、键盘方向键，双击恢复 50:50，窄屏自动切换单栏
- 默认深色主题，可切换浅色；仅主题偏好写入 `localStorage`
- 选择或拖放 `.md`、`.markdown`、`.txt` 文件
- 显示文件名、字符/行数和未保存修改状态，离开页面前提示
- 复制 Markdown 源文和清空内容
- 根据 `h1`～`h6` 自动生成可折叠目录，点击平滑跳转
- 支持标题、段落、粗体、斜体、删除线、链接、图片、行内代码、围栏代码块、引用、列表、任务列表、分隔线和表格

## 安全与隐私

- 正文只保留在当前标签页内，不写入 `localStorage` 或其他持久化存储
- Markdown 中的原生 HTML 全部转义，不创建或执行脚本
- 链接拒绝 `javascript:`、`vbscript:`、`data:`、`file:`、`blob:` 等危险协议
- 图片支持 PNG/JPEG/GIF/WebP 的 Base64 data URL，也可预览 HTTP(S) 图片
- 本地相对图片不会读取，预览区会显示明确提示，建议转换为 data URL
- 不加载 CDN、字体、脚本、样式或其他运行时依赖

## 使用

直接双击 `markdown-editor.html`，或在 macOS 终端执行：

```bash
open -a "Google Chrome" markdown-editor.html
```

选择文件或拖入编辑区域即可打开。编辑后，本工具不会覆盖原文件。桌面双栏下可拖动中间分隔条调整源文与预览宽度；聚焦分隔条后用左右方向键按 1% 调整，按住 Shift 时每次调整 5%，Home/End 可跳到 20%/80%，双击恢复 50:50。比例只保留在当前页面状态，不写入本地存储。

## 说明

这是轻量常用 Markdown 实现，不以完全兼容 CommonMark/GFM 的所有边缘语法为目标。列表以常见单层写法为主；表格分隔行至少需要三个连字符。远程 HTTP(S) 图片只有在网络可用时才能显示，离线文档建议使用 data URL。
