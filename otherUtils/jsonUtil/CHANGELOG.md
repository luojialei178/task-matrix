# JSON 格式化工具 - 开发记录

## 版本历史

### v1.1.0 - 2025-01-XX

#### 增强功能
- **左右联动高亮**：光标在左侧输入框移动时，右侧输出对应的 key/value 自动高亮（黄色背景），高亮持久化不消失
- **持久化括号高亮**：点击任意括号，该括号和配对括号高亮显示（蓝色背景），不再自动消失
- **互斥高亮模式**：
  - 点击数据节点时清除括号高亮，只显示数据高亮
  - 点击括号时清除数据高亮，只显示括号高亮
  - 左侧光标移动时清除括号高亮，触发数据联动高亮
  - 点击输出区空白处清除所有高亮

#### 技术改进
- 重写 `highlightJSON` 函数，使用逐字符解析替代正则表达式，提升可靠性
- 优化字符串查找算法，正确识别 key 和 value
- 添加 `json-data` class 标记所有数据节点，支持点击高亮

---

### v1.0.0 - 2026-07-01

#### 初始版本
- 单文件 HTML 实现，零依赖
- 核心功能：格式化、压缩、转义、反转义
- 快捷键：Cmd/Ctrl + Enter 快速格式化

#### 主题系统
- 深色/浅色双主题
- 标题右侧小圆点切换（深色模式白点，浅色模式黑点）
- 主题偏好自动保存到 localStorage（key: `json-formatter-theme`）
- macOS 风格毛玻璃效果

#### 语法高亮
**柔和配色方案（参考 VS Code / IDE）：**

深色主题：
- Key（键）：金色 `#d4a76a`
- String（字符串）：淡绿 `#a5d6a7`
- Number（数字）：淡蓝 `#9ab4f5`
- Boolean（true/false）：淡紫 `#ce93d8`
- Null：淡红 `#ef9a9a`

浅色主题：
- Key：深金 `#b8860b`
- String：深绿 `#2e7d32`
- Number：深蓝 `#1565c0`
- Boolean：深紫 `#7b1fa2`
- Null：深红 `#c62828`

#### 括号匹配
- 点击任意 `{` `}` `[` `]` 括号自动高亮配对括号
- 蓝色背景标识，闪烁动画 0.4 秒后自动取消
- Hover 时括号有背景提示

#### 交互优化
- 浅色模式面板边框加深到 `rgba(0,0,0,.12)`，提升视觉清晰度
- textarea 边框 `rgba(0,0,0,.15)`
- 面板添加微妙阴影 `box-shadow: 0 1px 3px rgba(0,0,0,.04)`
- 输出区域使用 `<div>` 替代 `<textarea>` 以支持富文本高亮

## 技术栈

- **运行环境**：浏览器（Chrome 125+）
- **依赖**：零外部依赖
- **存储**：localStorage（仅主题偏好）
- **样式**：CSS 变量 + 双主题 class（`body.dark` / `body.light`）

## 已知限制

- 输出区域不可直接编辑（只读显示）
- 括号匹配依赖预处理，超大 JSON（>1MB）可能有性能影响
- 转义/压缩模式不显示语法高亮（纯文本输出）
