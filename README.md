# ⊞ 任务优先级矩阵

一个零依赖的单文件桌面小工具，帮助你用二维矩阵管理任务优先级。双击 `task-matrix.html` 用 Chrome 打开即可使用，无需安装任何环境。

## 截图预览

> 深色 / 浅色双主题，macOS 毛玻璃风格

## 功能特性

### 二维矩阵
- 纵轴：任务优先级 P1（最高）→ P5（最低）
- 横轴：事项优先级 P1（最高）→ P5（最低）
- 单元格背景为热力图配色——行列优先级越高，颜色越深

### 任务卡片
- 点击单元格内 **＋ 添加任务**，回车确认，Esc 取消
- 左侧复选框：点击切换完成状态，完成后文字加删除线、卡片变绿
- 右侧 × 删除按钮：即时删除
- 任务卡片背景继承所在单元格的颜色色调
- 双击卡片文字可原地编辑

### 拖拽
- **任务卡片**：可拖拽到任意其他单元格
- **行标题**：拖拽行标题可交换两行数据，颜色按视觉位置固定不动

### 动态扩展行列
- 矩阵右侧 / 下方各有 **＋** 按钮，添加扩展列/行（灰色，标签为"扩展1""扩展2"…）
- 扩展行/列右上角有 × 删除按钮，删除后后续行列自动补位

### 行标题编辑
- 双击任意行标题进入内联编辑模式，回车或失焦保存
- 支持用业务语义替代 P1~P5（最多 12 字符）

### 主题切换
- Header 右侧小圆点切换深色 / 浅色
- 深色默认，macOS / iOS 系统色板，偏好自动保存

### 数据管理
- **自动保存**：所有数据实时写入 `localStorage`，刷新不丢失
- **导出 JSON**：将任务、行名称、行顺序一次性导出为 `task-matrix.json`
- **导入 JSON**：选择之前导出的文件，确认后完整还原数据
- **清除已完成**：一键批量删除所有已勾选完成的任务

## 使用方法

```bash
# 直接用 Chrome 打开即可
open -a "Google Chrome" task-matrix.html
```

无需 Node.js、无需构建、无需网络。

## 数据存储

所有数据存储在浏览器 `localStorage` 中，以下 key 对应不同数据：

| Key | 内容 |
|-----|------|
| `task-matrix-v6` | 任务数据（任务列表、扩展行列数量） |
| `task-matrix-theme` | 主题偏好（`dark` / `light`） |
| `task-matrix-rownames` | 自定义行标题名称 |
| `task-matrix-roworder` | 行排列顺序（拖拽换位后生效） |

> 如需跨设备同步或备份，使用**导出 JSON** 功能。

## 技术说明

- 单文件 HTML，所有 CSS / JS 内联，零外部依赖
- 兼容 Chrome 125+
- 样式使用 CSS 变量 + `body.dark` / `body.light` 双主题 class
- 数据结构版本：`v6`（破坏性改动时版本号递增）

## 文件说明

```
task-matrix.html    # 工具本体，唯一产出文件
task-matrix.json    # 导出的数据备份（可选）
```
